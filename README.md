# Bridgevalue Technology Services (BTS)

This repository contains the static web assets for Bridgevalue Technology Services Inc. (BTS), including its main site and product pages.

## Products

- **Atlas**: Time, Projects & Billing management tool.
- **Sentry**: Legal Document Comparison tool.

## Structure

- `bts-*.html`: Main corporate pages for BTS (About, Consulting, People, Contact, Index).
- `atlas-*.html`: Landing and feature pages for the **Atlas** product.
- `sentry-*.html`: Landing and feature pages for the **Sentry** product.
- `shared.css`: Central styles used across all pages.
- `shared.js`: Shared interactive behaviors and navigation script.
- `color-themes.html`: Theme references and visual styling assets.
